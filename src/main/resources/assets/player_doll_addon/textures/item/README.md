# 纹理文件说明

## 需要的纹理文件

请在此目录下创建以下纹理文件：

1. **player_doll.png** (16x16 或 32x32)
   - 玩家玩偶物品的图标
   - 建议：使用玩家头像或玩偶图标

2. **player_doll_creator.png** (16x16 或 32x32)
   - 玩家玩偶创建器物品的图标
   - 建议：使用工具或创建器图标

## 临时解决方案

如果暂时没有纹理文件，可以：
1. 使用原版物品纹理作为占位符
2. 或者创建简单的单色纹理

## 纹理文件位置

- `textures/item/player_doll.png`
- `textures/item/player_doll_creator.png`
